## 项目介绍

SmartBus 是一款智慧公交运行管理系统，提供公交线路、站点、车辆、司机等基础数据的综合管理功能，支持公交运行日常调度、实时运行状态监控可视化分析以及数据统计，旨在借助信息化手段提高公交运营效率，实现智能化调度与管理。

**适合毕设，课设，学习使用！**

## 主要功能模块

- **系统与权限管理**：用户管理、角色管理、菜单权限管理、操作日志
- **公交基础信息管理**：线路管理、站点管理、车辆管理、司机管理
- **线路站点关联编排**：站点停靠顺序配置
- **运行调度管理**：排班调度、出车/收车管理
- **实时监控大屏**：车辆实时定位、线路运行追踪、运营统计报表
- **智能运营管理**：线路运营健康状况评估、自动班次生成

## 技术架构

- 后端：Spring Boot 2.6 + MyBatis Plus + Spring Security + JWT
- 前端：Vue 3 + TypeScript + Vite + Element Plus + Pinia
- 数据库：MySQL 8.x
